## Overview

WatermelonDB provides many advanced features for complex applications:

- **Migrations** - Safely evolve your schema without losing user data
- **Local Storage** - Simple key/value store for small data
- **Type Safety** - Full Flow and TypeScript support
- **Performance** - Lazy loading and query optimization
- **Observability** - Reactive UI with automatic re-rendering
